Build & Run

From git clone to a running terminal in under 90 seconds on a warm Cargo cache. This page is your operator manual — system requirements, build commands, themes, keybindings, configuration, troubleshooting.

Requirements

Rust
2024 edition
GPU
OpenGL 3.3+ / Vulkan / Metal / D3D12
OS
macOS 10.15+ · Linux · Win 10+
Memory
4 GB+

Build

# clone
git clone https://github.com/jdmiranda/phantom.git
cd phantom

# build all binaries (release)
cargo build --release

# build specific crate
cargo build -p phantom-renderer

# run tests
cargo test

# run with logging
RUST_LOG=debug cargo run --bin phantom

Run

# recommended — supervisor + app together
cargo run --bin phantom-supervisor
# or just
./run.sh

# standalone (no supervisor, useful for hard-crash debugging)
cargo run --bin phantom

# theme variants
cargo run -- --theme amber
cargo run -- --theme ice
cargo run -- --theme blood
cargo run -- --theme vapor
cargo run -- --theme verdant
cargo run -- --theme plain    # disables CRT effects entirely

# skip the cinematic boot
cargo run -- --no-boot

# debug mode (more logging, debug overlays)
cargo run -- --debug

# GPU diagnostic dump
cargo run -- --gpu-info

Configuration

Phantom reads ~/.config/phantom/config.toml on boot. All fields optional.

[renderer]
theme        = "phosphor"     # phosphor · amber · ice · blood · vapor · verdant · plain
font_size    = 14.0
scanlines    = 0.8
bloom        = 0.6
curvature    = 0.1
chromatic    = 0.02
vignette     = 0.4
noise        = 0.05

[terminal]
shell             = "/bin/zsh"
working_directory = "~"
scrollback_lines  = 10000

[agents]
claude_api_key  = "sk-..."          # or via ANTHROPIC_API_KEY env
max_concurrent  = 3
timeout_seconds = 30

[brain]
enabled             = true
self_improvement    = false             # opt-in only
trust_band          = "Conservative"
audit_log_path      = "~/.phantom/audit/brain.jsonl"

Running the autonomous loops

# the four canonical loops against the phantom repo
phantom loop run \
    --repo  ~/code/phantom \
    --loops pr_finder_review,pr_finder_impl,reviewer,implementer

# the loop overseer ships preflight checks:
#   - gh binary must be on PATH
#   - gh auth status must report logged-in
#   - MCP tool names must not collide with complete_task / abort_task

# acquire an exclusive lock at .phantom/loops/.runlock to prevent races
# Ctrl-C releases the lock cleanly via Drop

Running phantom as a builder

# point phantom at any GitHub repo and let it eat the issue queue
phantom builder run \
    --target      your-org/your-repo \
    --trust-band  conservative \
    --max-prs     4

Hot shader reload

# shaders/crt.wgsl is polled on mtime change
PHANTOM_HOT_SHADERS=1 cargo run --bin phantom-supervisor

Troubleshooting

Build issues

# clean rebuild
cargo clean && cargo build

# refresh dependencies
cargo update

# check Rust version
rustc --version          # needs nightly with 2024 edition

Runtime issues

# full debug log
RUST_LOG=phantom=debug cargo run

# gpu capability dump
cargo run -- --gpu-info

# disable post-fx if GPU driver is flaky
cargo run -- --theme plain

# skip boot if iterating quickly
cargo run -- --no-boot

Diagnostics

# tail supervisor crash forensics
tail -f ~/.config/phantom/supervisor_events.jsonl | jq

# last app signal (SIGSEGV / SIGBUS / SIGTERM)
cat ~/.config/phantom/signal_crash.log

# brain audit log
tail -f .phantom/brain/audit.jsonl | jq

Common problems

SymptomLikely causeFix
Black screen after bootGPU adapter selected fallback backend--gpu-info, update drivers, try --theme plain
Glyphs missingSystem font not installedInstall JetBrains Mono or override in config
"capability denied"Agent's role doesn't have the tool's capability classWorking as intended — see agents
Loop refuses to start: "gh binary not found"gh CLI not installedbrew install gh + gh auth login
Loop refuses to start: "MCP tool collision"An MCP server registered a tool named complete_taskRename in the MCP server's manifest
Self-improvement won't enqueueTrust band threshold, rate limit, or hard exclusionCheck brain/audit.jsonl — every skip has a reason

Performance

  • GPU: Use the dedicated card on hybrid systems. --theme plain on integrated GPUs.
  • Glyph atlas: Grows on demand. Large terminals with mixed scripts may benefit from a larger preallocation.
  • Scrollback: Default 10K lines. Reduce for low-RAM hosts.
  • Agent concurrency: Default 3. Bump to taste, but each agent is a tokio-task footprint.
